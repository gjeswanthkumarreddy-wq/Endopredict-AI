# EndoPredict-AI

AI-powered Endometriosis Prediction System using Flask, PyTorch, CNN, and Clinical Data Analysis.

## Features

* Endometriosis risk prediction
* Clinical data analysis
* CNN + Numerical feature integration
* Flask backend API
* Interactive dashboard
* Analytics and reports
* Multi-language support

## Technologies Used

* Python
* Flask
* PyTorch
* HTML
* CSS
* JavaScript
